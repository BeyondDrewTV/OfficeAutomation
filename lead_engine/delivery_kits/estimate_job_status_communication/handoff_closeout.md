## Revision Boundary

**What is in scope without a change order:**
- Updating copy in an existing approved template (e.g., changing the phone number, tweaking the wording)
- Adding or removing a client name or detail field
- Adjusting timing on a single stage (e.g., send day-before instead of same-day)

**What requires a change order:**
- Adding a new status stage not covered in the original status map
- Adding a new send channel (e.g., adding email when only SMS was set up)
- Integrating with a new tool or platform
- Building any automation where only manual send was originally configured
- Building any client-facing portal, approval form, or response capture

Contact Drew to discuss before starting any out-of-scope change.
