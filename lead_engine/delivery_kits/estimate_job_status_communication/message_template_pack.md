# Estimate & Job Status Communication — Message Template Pack
Version: v1 | Kit: estimate_job_status_communication_v1

## How to Use This Pack
These are starter templates for each status stage. The owner must fill in the blanks, read each message, and check the approval box before any template goes live. Drew sets up the send path, but all copy must be owner-approved.

**Scope note: Every message goes out under the owner's name and contact, not Drew's. Drew builds the flow. The owner is the sender identity.**

Do not activate any template until the owner has checked "Approved" for that stage.
