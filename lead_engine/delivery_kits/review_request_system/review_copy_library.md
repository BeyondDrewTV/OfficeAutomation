# Review Request System — Review Copy Library
Version: v1 | Kit: review_request_system_v1

## How to Use This Library
The owner reviews and approves every template before any review request is sent. No template goes live without the approval checkbox checked.

Fill in each [placeholder] with real business details during setup. The owner reviews the final draft — not the placeholder version.

**Compliance note:** Review requests must be sent only to real, satisfied clients. Never incentivize reviews (no discounts, gift cards, or promises offered in exchange). Never send a review request to a client who expressed dissatisfaction. If a client seems unhappy, handle the concern directly — do not send a review request.
