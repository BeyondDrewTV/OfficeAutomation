# Basic Cleanup — Activation Checklist
Version: v1 | Kit: basic_cleanup_v1

This checklist runs a Basic Cleanup engagement. It is a narrowed-down path through the Presence Refresh kit. Do not perform any step marked out of scope in `bundle_scope_matrix.md`.

---

## Pre-Work

### Asset Request
Use the Presence Refresh asset request template: `lead_engine/delivery_kits/presence_refresh/asset_request_template.md`

Send it as-is. All required fields apply. The narrower scope does not reduce the asset requirement — you still need accurate business info before touching anything.

Required assets before work starts:
- ☐ Business name, phone, address, hours
- ☐ Services list (top 3–5)
- ☐ Service area confirmed
- ☐ At least one photo confirmed available (or owner confirms none available)
- ☐ GBP listing located (URL confirmed or found by Drew)

### Access
- ☐ GBP access confirmed (owner has granted access or handed off credentials)
- ☐ Facebook: read-only review only — no edit access required for this tier

Do not request Facebook edit access for a Basic Cleanup engagement. That is a Full Presence Refresh access requirement.

---

## Work Steps

Follow `lead_engine/delivery_kits/presence_refresh/presence_cleanup_checklist.md` and complete only the items marked "In scope" in `bundle_scope_matrix.md`.

In-scope work steps:
- ☐ GBP: verify and update name, phone, hours, address
- ☐ GBP: update description
- ☐ GBP: update services list
- ☐ GBP: review existing photos, update one if needed
- ☐ Facebook: review for obvious inconsistencies (read only — flag, do not fix)

Do not perform:
- GBP full photo upload
- GBP Q&A cleanup
- Facebook refresh or edits
- Cross-platform consistency fixes (flag only)
- Trust copy rewrite

If a client requests any out-of-scope item during the engagement, document it as a flagged item and present it as a change order or upgrade path at closeout.

---

## QA

Use `lead_engine/delivery_kits/presence_refresh/presence_qa_checklist.md` and skip the out-of-scope items.

In-scope QA checks:
- ☐ GBP name, phone, hours, address — correct and live
- ☐ GBP description updated and published
- ☐ GBP services list updated and published
- ☐ GBP photo updated (or confirmed no update needed)
- ☐ Facebook inconsistencies flagged in closeout

Skip:
- GBP Q&A, full photo refresh, cross-platform fix verification, trust copy review

---

## Closeout

Use `lead_engine/delivery_kits/basic_cleanup/closeout_template.md`.

Do not use the Presence Refresh closeout snapshot template for a Basic Cleanup — the scope ceiling and upgrade path language in the Basic Cleanup template are required.

- ☐ Closeout template filled in
- ☐ Out-of-scope items listed (flagged but not addressed)
- ☐ Upgrade path offered
- ☐ Owner acknowledged scope covered
- ☐ Engagement closed in Copperline
