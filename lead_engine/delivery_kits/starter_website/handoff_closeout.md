## Revision Boundary

**One standard revision round is included in the project scope.** This covers:
- Minor copy corrections
- Small layout tweaks
- Photo swaps or updates
- CTA destination changes

**What is not included in the standard revision:**
- New sections (e.g., adding a FAQ section, a new service page, a gallery page)
- Adding a second page to the site
- Redesigning the layout or color scheme
- Building new functionality (booking forms, live chat, portals)

Any of the above is a change order. Drew will provide a quote before starting any out-of-scope work.
